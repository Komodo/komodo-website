System Requirements

Windows

  • Windows 7 or later, including Windows 8
  • 1GHz (or faster) x86 or x86_64 processor
  • 1 GB RAM
  • 250 MB hard disk space
  • 350 MB of temporary hard disk space during installation

Mac

  • Mac OS X 10.9 (Mavericks) or later
  • Intel processor (x86_64 only)
  • 1GB RAM
  • 250 MB hard disk space
  • 350 MB of temporary hard disk space during installation

Linux

  • 1GHz (or faster) x86 or x86_64 processor
  • 1 GB RAM
  • 250 MB hard disk space
  • 350 MB of temporary hard disk space during installation
System Libraries:
  • glibc 2.4 (or higher) and libjpeg.so.62 (or higher)
  • libstdc++6 (or higher)
  • GTK+ 2.10 (or higher)
  • Gnome (libgnome, libgnomeui)
  • gdk-pixbuf2 may be required on some distros (eg. CentOS 6)

The following Linux distributions satisfy these requirements and are officially supported:

  • Red Hat Enterprise Linux 5 or later
  • CentOS 6.0 or later (see requirements)
  • Fedora Core 18 or later
  • OpenSUSE 13.1 or later
  • Ubuntu 12.04 or later
  • SuSE Linux Enterprise Desktop/Server 11.2 or later
